# 00. Assignment Mindset

- This is a product-engineer backend assignment, not a generic CRUD demo.
- Prefer a small complete system over broad unfinished features.
- Show judgment through requirements interpretation, tests, README, and trade-off explanations.
- AI may help, but final code, validation, and README must reflect the user's understanding.
- Make the reviewer able to run, inspect, and trust the project quickly.
